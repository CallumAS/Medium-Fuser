## ADD USER CONTENT

Example: https://medium.com/@NatashaMH => http://localhost:3003/?name=NatashaMH

ADDS THE USERS CONTENT TO SCRAPE

## VIEW SCRAPED CONTENT

http://localhost:3003/data
